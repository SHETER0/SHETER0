# Project Overview

This project is a **personal portfolio website** for Faisal Ateeq Al-Mutairi, showcasing his work as a Software Developer, Game Developer, and Full-Stack Engineer.

## Features
- **Responsive Design**: Built with React, Tailwind CSS, and Vite.
- **Multi-language Support**: Integrated i18n for international users.
- **Interactive UI**: Includes various components like accordions, buttons, and navigation menus.
- **Project Showcase**: A dedicated section to display ongoing projects and skills.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Build Tool**: Vite
- **Icons/UI Components**: Lucide React, Shadcn UI
- **Deployment**: Optimized for Vercel/Netlify deployment.

## Getting Started
To run this project locally:
1. Clone the repository.
2. Install dependencies: `npm install` or `pnpm install`.
3. Start the development server: `npm run dev`.
